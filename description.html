HTML Terms

HTML
Hyper Text Markup Language, used to display text, images, links, and other things of the sort to the user.
HTML element
Elements are the building blocks of your website, like paragraphs, italic, bold, line breaks, or others of that kind.
HTML tag
The tags used to define where and what effects happen on your words. For example, this line is between a <dd> and </dd> tag.
angle bracket
"<" and ">" are angle brackets, used to enclose tags
HTML attribute
Attributes are modifiers given to tags to change how they function, such as choosing between a "1, 2, 3, 4, 5" list and a "i, ii, iii, iv, v" kind of list.
HTML comment
Comments are meant for people viewing the html to inform them what's what. To make them, use <!-- comment -->
HTML object element
Elements like <br> and <hr> are object elements, they don't contain any text in them, they just exist on their own
HTML special character
Special characters are things like <, >, and &. These characters have to be entered using special sequences of characters in HTML. Here is how you make each one:
  • &lt; --> "<"
  • &gt; --> ">"
  • &amp; --> "&"
  • &nbsp; --> non-breaking space
